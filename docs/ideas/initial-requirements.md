# Pomodoro App Initial Requirements

最終更新: 2026-04-08

## 目的

集中作業と休憩を繰り返せる、シンプルなポモドーロアプリを作る。

## 想定ユーザー

- 個人で作業時間を管理したい人
- 長いタスクを短い集中時間に分けたい人

## まず欲しい機能

- 25 分の作業タイマー
- 開始、停止、リセット
- 休憩への切り替え
- 何回完了したかの表示

## 次に欲しい機能

- 作業時間と休憩時間の変更
- セッション履歴の保持
- 日次の実績表示

## 制約

- まずはフロントエンド単体で進める
- 大きい構成変更より、既存の feature-based 構成を活かす
- 1 つ目の feature は timer を優先する

## 次の依頼例

```text
brainstorm でポモドーロアプリの initial-requirements を壁打ちして。
```

```text
setup-project でポモドーロアプリの初期 docs を整備して。
```

```text
feature-add でタイマー機能を追加して。
```

# 💻 アプリケーション概要

このアプリケーションは比較的シンプルな構成です。ユーザーはチームを作成して他のユーザーを招待し、チームメンバー同士でさまざまなトピックについてディスカッションを行うことができます。

登録時に既存チームへの参加を選ばなかった場合は、新しいチームが自動的に作成され、そのユーザーがチームの管理者（ADMIN）になります。

[デモ](https://bulletproof-react-app.netlify.app)

## データモデル

アプリケーションは以下のモデルで構成されています:

- **User** — 以下のロールのいずれかを持つ:

  - `ADMIN` ができること:
    - ディスカッションの作成・編集・削除
    - すべてのコメントの作成・削除
    - ユーザーの削除
    - 自分のプロフィール編集
  - `USER` ができること:
    - 自分のプロフィール編集
    - 自分のコメントの作成・削除

- **Team**: 1人の管理者と複数ユーザーで構成されるチーム。メンバー同士でディスカッションに参加できる。

- **Discussion**: チームメンバーが作成するディスカッション。

- **Comment**: ディスカッション内のすべてのメッセージ。
